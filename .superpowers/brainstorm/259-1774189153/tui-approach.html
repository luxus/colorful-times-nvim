A — Table Manager

A focused floating window showing the schedule as a navigable table. Keyboard-driven: navigate rows, press keys to add/edit/delete. Modal workflow for editing.

B — Step Wizard

Multi-step form: one field at a time, progress indicator, Tab/Enter to advance. Clean and guided. Great for first-time setup. Each field validates inline.

C — Split Panel Dashboard

Two-pane layout: schedule list on left, live edit form on right. Select an entry and edit fields directly. Status bar shows plugin state. Like a mini IDE panel.
